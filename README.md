# Stark

> “I am Iron Man” — _Tony Stark_

[Powerline] prompt based on Iron Man.

![Stark]

## Install

With [fisherman]

```fish
fisher stark
```

## Terminal Settings

* [Stark.terminal]
* [Stark.itermcolors]
* Font: 16pt Source Code Pro Medium
* Non-ASCII Font: 17pt Source Code Pro for [Powerline]
